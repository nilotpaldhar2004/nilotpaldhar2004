## 📦 Open Source — DataDiagnose

<div align="center">

[![PyPI version](https://img.shields.io/pypi/v/datadiagnose?style=for-the-badge&color=2563eb&label=PyPI%20%7C%20datadiagnose)](https://pypi.org/project/datadiagnose/)
[![PyPI Downloads](https://img.shields.io/pypi/dm/datadiagnose?style=for-the-badge&color=059669&label=Monthly%20Downloads)](https://pypi.org/project/datadiagnose/)
[![License](https://img.shields.io/badge/License-MIT-f97316?style=for-the-badge)](https://github.com/nilotpaldhar2004/datadiagnose/blob/main/LICENSE.txt)
[![Zero Dependencies](https://img.shields.io/badge/Dependencies-Zero-7c3aed?style=for-the-badge)](https://pypi.org/project/datadiagnose/)

</div>

**DataDiagnose** automatically audits ML datasets before training begins. One function call returns a full health report — no external dependencies required.

```bash
pip install datadiagnose
```

```python
from datadiagnose import DataDiagnose
import pandas as pd

df  = pd.read_csv("your_dataset.csv")
dd  = DataDiagnose(df)
report = dd.diagnose()

# Returns: 0–100 health score, detected issues, model type recommendations
```

**What it detects:**

| Check | Description |
|:------|:------------|
| Missing Values | Identifies columns and rows with null/NaN data |
| Duplicate Rows | Flags exact and near-duplicate records |
| Class Imbalance | Detects skewed target distributions |
| Data Leakage | Catches features that leak target information |
| High Cardinality | Warns on categorical columns with too many unique values |
| Skewed Features | Identifies heavily skewed numerical distributions |
| Outliers | Statistical outlier detection across all numeric columns |
| Constant Features | Removes zero-variance columns that add no signal |

**Zero external dependencies · Pure Python standard library · 140-test suite · MIT License**
